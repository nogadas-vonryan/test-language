## Language Constructs

`$var_name = value` = declares a variable

`fry value` = print

`prompt value` = print to prompt's output buffer

`pick var` = prompts for input

`oc(condition)` = if statement

`si(count)` = loop

`br` = prints newline

`clear` = clears output

`👏👏 code 👏👏👏` = defines a scope
